# Rust for fun

## Setup

### Installing rust

Via curl: 

```shell
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```
Or via Homebrew

```shell
brew install rust
```

### Upgrade

```shell
rustup update 
```

### Uninstall

```shell
rustup self uninstall
```

### check Rust version

```shell
rustc --version
```
